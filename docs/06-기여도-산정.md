# 06. 기여도 산정 공식

회의 기여도는 **3축(액션 / 발언 / 참석)** 으로 측정한다. 결과는 `contribution_scores` 테이블에 회의 단위로 1건 저장 (UNIQUE: `user_id` + `meeting_id`).

**대상**: `meetings.meeting_type = 'regular'` (정규 회의).

---

## 종합

```
종합 = (액션 × 0.5 + 발언 × 0.3 + 참석 × 0.2) / 적용 가중치 합
composite_score
```

→ `contribution_scores.composite_score`

각 축 = 두 지표의 단순 평균. 측정 불가 지표/축은 분모에서 제외하고 남은 가중치가 비례 확장된다.

---

## 액션 축

```
완료율 = 완료한 액션 / 전체 액션 (난이도 가중)
action_completion_ratio = Σ(done × difficulty) / Σ(difficulty)
```

→ `action_items` (status, difficulty)

```
마감 준수 = 액션별 마감점수의 난이도 가중 평균
deadline_adherence = Σ(마감점수 × difficulty) / Σ(difficulty)

마감점수:
    done && completed_at ≤ due_at      → 1.0
    done && ≤ 1일 초과                  → 0.8
    done && ≤ 3일                       → 0.6
    done && ≤ 7일                       → 0.3
    미완료 / 7일 초과 / cancelled        → 0.0
```

→ `action_items` (status, completed_at, due_at, difficulty)

```
액션 = (완료율 + 마감 준수) / 2
```

---

## 발언 축

```
발언 비중 = 본인 글자수 / 전체 참여자 글자수 (활성 윈도우 내)
speech_ratio = Σ(본인 char_count) / Σ(전체 char_count)
```

→ `utterances` (user_id, char_count, started_at_offset_ms)

```
발언 안정성 = 발언한 안건 수 / 진행된 안건 수
speech_consistency = 본인 발언 agenda 수 / 활성 중 진행된 agenda 수
```

→ `utterances.agenda_id`, `agendas`

```
활성 윈도우 = [첫 join, 마지막 leave] - 비자발 끊김 - STT 실패
```

→ `presence_events.disconnect_classification`, `anomaly_events.event_type = 'stt_failure'`

> STT 실패 판정 방법은 미정 — [09](09-미결정-사항.md) 참조.

```
발언 = (발언 비중 + 발언 안정성) / 2
```

---

## 참석 축

```
출석 = (실제 참여 시간) / (전체 시간 - 비자발 끊김)
attendance_ratio = (참여 - 자발 자리비움 - 비자발 끊김) / (total_minutes - 비자발 끊김)
```

→ `presence_events`, `meetings.total_minutes`

```
정시 도착 = 1 - 지각 분 / 허용 지각
punctuality_score = max(0, 1 - 지각 분 / (total_minutes × 0.10))
```

→ `presence_events` (첫 join), `meetings.t0_timestamp`, `team_settings.punctuality_grace_ratio`

```
참석 = (출석 + 정시 도착) / 2
```

---

## 누적 점수 (여러 회의 기반 총 점수) — 당근 온도와 비슷

```
누적 = Σ(회의 점수 × 회의 시간) / Σ(회의 시간)
cumulative = Σ(composite_score × total_minutes) / Σ(total_minutes)
```

→ `contribution_scores.composite_score`, `meetings.total_minutes`

---

## 신규 필드 (06 기준으로 참조, 03 데이터 모델 미반영 — [09](09-미결정-사항.md))

| 위치 | 필드 | 용도 |
| --- | --- | --- |
| `meetings` | `meeting_type` (`regular` / 기타) | 정규 회의만 기여도 산정 대상 |
| `action_items` | `difficulty` (정수 1~3: 쉬움/보통/어려움) | 액션 완료율·마감 준수의 가중치 |
| `presence_events` | `disconnect_classification` (자발 / 비자발) | 활성 윈도우·출석률 분모 보정 |
| `anomaly_events` | `event_type = 'stt_failure'` | 활성 윈도우에서 차감 |
