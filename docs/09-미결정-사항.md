# 09. 미결정 사항

구현 착수 전 또는 PoC 후 결정이 필요한 항목.

## STT 관련 (Web Speech API — MVP)

- [ ] `confidence` 산출 방식 — `SpeechRecognitionAlternative.confidence` 신뢰성(브라우저별 0/미제공 케이스) 확인 후 보정·대체식 결정
- [ ] **STT 실패 판정 방식** — 활성 윈도우에서 차감하는 `anomaly_events.event_type='stt_failure'`의 트리거 조건 (재시작 공백·`network` 오류·연속 결손 구간 등)
- [ ] 지원 브라우저 범위·미지원 안내 정책 (Chrome/Edge 중심, Safari·Firefox 차이)
- [ ] ~60초 강제 종료 자동 재시작 시 발생하는 공백 구간 손실 처리 기준

## 도메인·HTTPS (웹 전환에 따른 신규)

- [ ] 도메인 발급·DNS 운영 주체 — 외부 등록처 + EC2 Caddy/Let's Encrypt 기준 ([10](10-AWS-인프라-제약.md))
- [ ] 프론트 정적 서빙 방식 — EC2 Caddy 단일 서빙(기본) vs Amplify 분리

## V2 (데스크탑 전환 후)

- [ ] **faster-whisper 모델 크기 선택** (`tiny` / `base` / `small`) — RealtimeSTT 도입 시 한국어 정확도·지연 비교 후 결정
- [ ] VAD 임계값 (RealtimeSTT 내장 VAD 파라미터)
- [ ] faster-whisper 모델 가중치 다운로드 호스팅 위치·무결성 검증 방법

## 데이터 모델 후속 반영 (06 공식 반영분)

06 새 공식 및 2026-05-29 정책 결정이 도입한 신규 필드를 03 데이터 모델에 반영해야 함:

- [ ] `meetings.meeting_type` (`regular` / `partial`) — 정규 회의만 기여도 산정 대상, 부분 회의는 누적 미반영
- [ ] `meetings.is_invalidated` (불리언) — 팀장 수동 무효 처리 시 누적·기여도 제외
- [ ] `meetings.duration` 기반 10분 미만 누적 미반영 처리
- [ ] `action_items.difficulty` (정수 1~3: 쉬움/보통/어려움) — 액션 축 가중치
- [ ] `action_items.due_date` 기본값 = 다음 정규 회의 시작 시각
- [ ] `presence_events.disconnect_classification` (자발 / 비자발) — 활성 윈도우·출석률 분모 보정
- [ ] `presence_events`에 사유결석 사유·팀장 승인 상태 필드 — 사유결석(승인분)만 누적 분모에서 제외
- [ ] `anomaly_events.event_type = 'stt_failure'` — 활성 윈도우 차감 트리거
- [ ] `attachments` 테이블 — 액션 아이템에 첨부되는 자료 (S3 키·파일명·MIME·크기)
